## 🏁 How to Run terminal-mario-kart on Windows

1. Make sure you installed Node.js first. To check, open Command Prompt and type:  
   `node -v`  
   It should display a version number (like v16.0.0). If it shows an error, install Node.js from https://nodejs.org/.

2. Open the **Command Prompt** app:  
   - Press `Windows + R` keys.  
   - Type `cmd` and press `Enter`.

3. In the Command Prompt window, navigate to the folder where you extracted the game files. Use the `cd` command to change directories. For example, if you extracted to Desktop:  
   `cd Desktop\terminal-mario-kart-master`

4. Once inside the game folder, type the following command to install the required files:  
   `npm install`  
   This command downloads all the small building blocks the game needs to run.

5. After the installation is complete, start the game by typing:  
   `node main.js`  

6. The game will launch inside the terminal window. Follow the on-screen instructions to pick your game mode and start racing.
